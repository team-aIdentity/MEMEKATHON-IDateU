# 개발 노트

## 📋 프로젝트 개요

MemeCore 체인 블록체인 해커톤을 위한 MEMEX API 기반 Social-Fi 데이팅 앱입니다.

## 🎯 핵심 기능 아이디어

### 1. 사용자 매칭 시스템
- MEMEX API의 사용자 정보를 활용한 매칭
- 토큰 보유량, 활동량 등을 기반으로 한 매칭 알고리즘
- 블록체인 기반 신뢰도 시스템

### 2. 소셜 피드
- MEMEX API의 게시물 피드 활용
- 좋아요, 리포스트, 댓글 기능
- 실시간 업데이트

### 3. 토큰 기반 인센티브
- MemeCore 네트워크의 MRC-20 토큰 활용
- 매칭 성공, 활동 참여 시 토큰 보상
- 토큰 스테이킹 기능

### 4. 채팅 기능
- 매칭된 사용자 간 1:1 채팅
- 블록체인 기반 메시지 암호화 (선택사항)

### 5. 프로필 관리
- MEMEX API의 사용자 프로필 활용
- 커스텀 프로필 이미지, 바이오
- 토큰 홀더 정보 표시

## 🔧 기술적 고려사항

### API 통합
- JWT 기반 인증 처리
- Cursor 기반 페이징 구현
- 에러 핸들링 및 재시도 로직

### 블록체인 통합
- MemeCore 네트워크 연결
- Web3 지갑 연동 (MetaMask 등)
- 스마트 컨트랙트 호출 (필요시)

### 데이터 관리
- 사용자 상태 관리
- 캐싱 전략
- 오프라인 지원 (선택사항)
